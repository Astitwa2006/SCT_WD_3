# SCT_WD_3: Quiz Game Application

**SkillCraft Technology Web Development Internship — Task 03**

An interactive 10-question web development quiz featuring a countdown timer, live score tracking, and a brutalist design.

## Features
- **Brutalist Aesthetic:** Solid borders, stark colors, and geometric layouts.
- **Countdown Timer:** 30 seconds per question with a visual progress ring.
- **Live Scoring & Progress:** Tracks your score and progress bar in real-time.
- **Results Screen:** Displays your final score, percentage, and a dynamic 3-star rating system.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
